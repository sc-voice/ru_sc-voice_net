Create a new EBT-Site

Copy this template to create your own EBT Site

Overview

To create your own EBT-Vue3 website, you'll need a Github account (Note: a free account is fine).

Create an Organizational Account

Websites are normally collaborative, so your EBT-Site will need a Github organizational account for your new EBT-Site. An organizational account will help you work with others on content creation.

Account Settings

EBT-Sites rely on Github Actions for updating content and software. Navigate to your Github account and click Settings In the left navbar, open the Actions panel. Then select General and scroll down to Workflow Permissions. Update workflow permissions as follows:

Click Save to update your workflow permissions.

Create new repository

  1. Navigate to sc-voice on Github
  2. Login to Github
  3. Then click the Use this template button, and choose Create new repository.

.

Configure new repository

After clicking Create new repository, you'll see a new webpage form for you to fill out:

  1. Specify the new repository name (e.g., "my-ebt-site")
  2. Choose Public repository (free)
  3. Select Include all branches
  4. Review your form entries
  5. When ready, click Create repository from template

We now need to enable Github pages for your new repository.

Enable Github pages
