Tutorial

Learn how to use ru.sc-voice.net

ru.sc-voice.net позволяет Вам читать и слушать ранние буддийские тексты (EBT). содержание с сайта SuttaCentral.net. ru.sc-voice.net показывает контент, используя одну или несколько карт:

  • Эта страница находится на Вики-карте.
  • Поисковые карты позволяют Вам искать содержимое EBT.
  • Карточки сутты позволяют Вам читать/воспроизводить содержание EBT.

В этом уроке Вы узнаете, как:

1. Закрытие карты
  • Нажмите на "-", чтобы свернуть карточку.
  • Нажмите на "X", чтобы удалить карту.
2. Откройте карту

У каждой карты есть вкладка карты в панели вкладок карты. Нажмите на вкладку карты, чтобы показать эту карту.

3. Поиск содержимого EBT Чтобы найти материалы EBT:
  • Введите фразу (например, "корень страданий")
  • Введите идентификатор сутты (например, "thig1.1") в текстовое поле поиска.
  • Нажмите "Inspire Me" для случайной поисковой фразы.

Просканируйте результаты поиска и нажмите на любой результат, чтобы увидеть настоящую сутту.

4. Воспроизводите содержимое EBT. Если текущая карта - это сутта, Вы можете:
  • Нажмите на любой сегмент
  • Воспроизведите выбранный сегмент
  • Доиграйте до конца сутты
5. Персонализация настроек

Настройки сгруппированы по разделам. Щелкните по каждому разделу и изучите свои возможности.
